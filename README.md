# Django Login and Registration Form

This project is a simple Django application that provides user authentication functionality, including user registration, login with email verification, logout, and reset password.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/django-login-registrationform.git
    cd django-login-registrationform
    ```

2. Create and activate a virtual environment:
    ```bash
    python3 -m venv venv
    source venv/bin/activate 
    ```

3. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Apply migrations:
    ```bash
    python manage.py migrate
    ```

5. Run the development server:
    ```bash
    python manage.py runserver
    ```
